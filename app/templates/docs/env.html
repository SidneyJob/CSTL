Environment variables

Переменные Описание
HOST Переменная для Flask. Ставь 0.0.0.0, если лаба должна быть доступна с других хостов в сети, или 127.0.0.1, если доступ должен иметь только локальный хост.
PORT На каком порту запуститься сервер. Пока работает только при запуске без Docker контейнера.
SECRET_KEY Секретный ключ для подписывания JWT токенов.
ATTACK_DOMAIN На какой домен будут идти запросы со страницы "CORS testing"
INTERACTSH_DOMAIN Куда слать ответы от сервера, при CORS мисконфигурациях
CORS_DOMAIN Домен для конфигурации CORS, позволяющий межсайтовое чтение со всех поддоменов, в рамках сессии пользователя
FLASK_DEBUG Включать ли режим дебага во Flask. К режиму дебага добавляется логирование событий уровня DEBUG.
FLASK_RUN_CERT Использовать ли SSL сертификаты (лежат в директории certs)
LOG_REQUESTS Логировать ли запросы к серверу
