Enviroment Settings

HOST: Определяет из какой сети будет доступа лаборатория
PORT: Определяет на каком порту будет работать лаборатория
SECRET_KEY: Определяет секретный ключ для подписи JWT токенов
ATTACK_DOMAIN: Определяет на какой домен будут идти запросы со страниц /cors_testing и /csrf_testing
INTERACTSH_DOMAIN: Определяет куда будет отправляться ответ от сервера при эксплуатации CORS мисконфигов
CORS_DOMAIN: Нужен для правильной настроки CORS. CORS настроен разрешать чтение в рамках сессии пользователя со всех поддоменов
FLASK_DEBUG: Определяет стандартную опцию `debug` во Flask
FLASK_RUN_CERT: Определяет будет ли использоваться SSL
LOG_REQUESTS: Определяет будут ли логироваться запросы


Types of cookies

1) SameSite-None: SameSite=None, HttpOnly=False, Secure=True
2) SameSite-Lax: SameSite=Lax, HttpOnly=False, Secure=False
3) SameSite-Strict: SameSite=Strict, HttpOnly=False, Secure=False
4) Undefined-SameSite: SameSite=[undefined], HttpOnly=False, Secure=False

5) HttpOnly-SameSite-None: SameSite=None, HttpOnly=True, Secure=True
6) HttpOnly-SameSite-Lax: SameSite=None, HttpOnly=True, Secure=False
7) HttpOnly-SameSite-Strict: SameSite=None, HttpOnly=True, Secure=False
8) HttpOnly-Undefined-SameSite: SameSite=[undefined], HttpOnly=True, Secure=False
